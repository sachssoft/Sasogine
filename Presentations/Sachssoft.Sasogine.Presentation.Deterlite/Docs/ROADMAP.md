# Roadmap

This roadmap outlines the planned development and future direction of the project.

## Rules for Deterlite

* AOT / Native
* No dynamic code generation (e.g., no `Reflection.Emit`)
* Performance-focused
* Deterministic behavior
* Cross-platform compatibility (Desktop, Mobile, Console, ...)

## Supported Platforms

* Sasogine (Engine + MonoGame)
* MonoGame

### Next Planned Platforms

* Other XNA-like frameworks (FNA, KNI)
* Unity
* Stride
* Godot
* ...
