# Roadmap

Diese Roadmap beschreibt die geplante Entwicklung und zukünftige Ausrichtung des Projekts.

## Regeln für Deterlite

* AOT / Native
* Keine dynamische Codegenerierung (z. B. kein `Reflection.Emit`)
* Fokus auf Performance
* Deterministisches Verhalten
* Plattformübergreifende Kompatibilität (Desktop, Mobile, Konsole, ...)

## Unterstützte Plattformen

* Sasogine (Engine + MonoGame)
* MonoGame

### Nächste geplante Plattformen

* Weitere XNA-ähnliche Frameworks (FNA, KNI)
* Unity
* Stride
* Godot
* ...
